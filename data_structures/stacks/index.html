    Stack (C++ STL)¶

    Stack nima?¶

    Stack ‚Äî bu C++ tilidagi ma‚Äôlumotlar tuzilmasi bo‚Äòlib, elementlarni LIFO tamoyili asosida saqlaydi.

    LIFO (Last In, First Out) ‚Äî oxirgi qo‚Äòshilgan element birinchi bo‚Äòlib olinadi.

    Tasavvur qilish uchun: pankeyklar (blinchiklar) uyumini o‚Äòylab ko‚Äòring üçΩÔ∏è

    • Pankeyk faqat tepadan olinadi
    • Oxirgi qo‚Äòshilgan pankeyk ‚Äî birinchi olinadi

    Dasturlashda aynan shu tamoyil Stack deb ataladi.

    Stack xususiyatlari¶

    • Elementlar faqat tepadan (top) qo‚Äòshiladi
    • Elementlar faqat tepadan (top) o‚Äòchiriladi
    • Index orqali murojaat qilish mavjud emas
    • Faqat eng yuqori (top) element bilan ishlash mumkin

    Qachon stack ishlatiladi?¶

    Stack quyidagi holatlarda juda foydali:

    • Funksiya chaqiruvlari (call stack)
    • Undo / Redo mexanizmlari
    • Qavslarni tekshirish ((), {}, [])
    • Backtracking algoritmlari
